# Epistack evidence appendix

This appendix contains the smallest public record needed to check the submission's central claims. It keeps unlike evidence separate: a source manifest is not a benchmark, a component test is not a scientific result, and a single product comparison is not evidence of general superiority.

## Evidence index

| Record | What it supports | Boundary |
|---|---|---|
| [COVID case](covid/README.md) | Ten-source corpus record; 69,774 canonical words; three preserved source-cited reviews; public source fingerprints | Designed source partitions; not independent expert opinion; does not resolve COVID origins |
| [Crux validation](crux/validation.md) | Runnable deterministic comparison; concrete claim, position labels, source references, dispute type, and change evidence; four tests | Structural and evidentiary validation, not general semantic accuracy |
| [LHC case](lhc/README.md) | Six-source, 78,481-word panel corpus; conditional safety map; documented authorship, advice, citation, and specialist-input relationships; 2-to-12 passage experiment | Not a new safety assessment; panel did not complete the relationship audit; no general automated relationship detector |
| [Perplexity comparison](perplexity/comparison.md) | One predeclared Deep Research call using the same question and starting-source list; complete returned answer; audited relationship correction | One instructed run; no typical-product or superiority claim |

## Runnable component

The public [Crux demonstration](../demo-crux/README.md) contains only standalone standard-library code, a sanitized real-panel fixture, and tests. It contains no proprietary Atlas implementation, internal services, database code, environment configuration, local paths, or secrets.

## Integrity

`SHA256SUMS.txt` records the SHA-256 of every public evidence and demo file except the checksum file itself. The COVID and LHC source manifests also retain fingerprints of the source texts used in the preserved runs, even though the full texts are not redistributed.

## Publication boundary

This appendix excludes:

- full third-party source texts whose redistribution rights were not established;
- proprietary Atlas code and internal configuration;
- raw service transcripts, internal logs, database details, and local paths;
- model-writing alternatives and internal reviews;
- working-folder version history; and
- redundant walkthroughs and superseded proposal files.

The omitted materials remain part of the private working record where applicable. Their omission means that this public appendix supports source identity, case counts, component behavior, and the stated documentary findings without publishing the entire research workspace.
